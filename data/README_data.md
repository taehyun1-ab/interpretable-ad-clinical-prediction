# Data

Raw ADNI and AIBL data are not included in this repository due to data use agreements.

Users must request access from the official data portals:

- ADNI: Alzheimer's Disease Neuroimaging Initiative
- AIBL: Australian Imaging, Biomarkers and Lifestyle Study of Ageing

Expected input files after preprocessing:

- `data/processed/adni_baseline_features.csv`
- `data/processed/aibl_shared_features.csv`

The primary ADNI experiments used baseline clinical variables only.  
The AIBL external validation used shared clinical variables available in both ADNI and AIBL.
