# Results

This directory is used to store model outputs, evaluation results, figures, checkpoints, and supplementary analysis files generated by the notebooks.

Generated result files are not included in this repository because they may be derived from restricted ADNI and AIBL data.

## Expected Output Structure

After running the notebooks, this directory may contain:

```text
results/
├── README_results.md
├── internal_test_results.csv
├── training_history.csv
├── external_aibl_results.csv
├── shared_variable_validation_summary.csv
├── checkpoints/
│   ├── AD_vs_CN_best.pt
│   ├── AD_vs_MCI_best.pt
│   ├── MCI_vs_CN_best.pt
│   └── pMCI_vs_sMCI_best.pt
├── figures/
│   ├── internal_roc_curves.png
│   ├── shap_summary_pMCI_vs_sMCI.png
│   ├── shap_waterfall_correct_pMCI_vs_sMCI.png
│   └── shap_waterfall_incorrect_pMCI_vs_sMCI.png

    ├── supplementary_table_s3_threshold_sensitivity.csv
    ├── supplementary_table_s4_seed_robustness.csv
    ├── supplementary_table_s6_feature_ablation.csv
```

## Main Result Files

### `internal_test_results.csv`

Generated by:

```text
notebooks/02_model_training_and_internal_validation.ipynb
```

This file contains internal ADNI held-out test performance for the four binary classification tasks:

- AD vs. CN
- AD vs. MCI
- MCI vs. CN
- pMCI vs. sMCI

Reported metrics include:

- AUC
- accuracy
- balanced accuracy
- precision
- sensitivity
- specificity
- F1-score

### `training_history.csv`

Generated by:

```text
notebooks/02_model_training_and_internal_validation.ipynb
```

This file stores epoch-wise training loss and validation AUC for each task.

### `external_aibl_results.csv`

Generated by:

```text
notebooks/03_external_validation.ipynb
```

This file contains shared-variable internal and external validation results using ADNI and AIBL.

AIBL samples are used only for external validation and are not used for:

- training
- validation
- hyperparameter tuning
- early stopping
- model selection

### `shared_variable_validation_summary.csv`

Generated by:

```text
notebooks/03_external_validation.ipynb
```

This file summarizes the main shared-variable ADNI/AIBL validation results for manuscript reporting.

## Checkpoints

Model checkpoints are saved under:

```text
results/checkpoints/
```

Each checkpoint may contain:

- model state dictionary
- selected hyperparameter configuration
- fitted scaler
- feature column names
- best validation AUC
- best epoch
- random seed

Checkpoint files are excluded from version control because they are generated from restricted ADNI-derived data.

## Figures

Figures are saved under:

```text
results/figures/
```

Expected figures include:

- internal ROC curves
- SHAP global summary plot
- SHAP local/waterfall explanation plots

## Supplementary Results

Supplementary analysis outputs are saved under:

```text
results/supplementary/
```

These files correspond to additional analyses reported in the Supplementary Material, including:

- threshold sensitivity analysis
- random seed robustness analysis
- feature ablation analysis
- permutation analysis
- latent-space visualization using t-SNE and UMAP

## Version Control Note

The following generated files and directories should not be committed to the public repository:

```text
results/*.csv
results/*.png
results/checkpoints/
results/figures/
results/supplementary/
```

Only this `README_results.md` file is tracked to describe the expected result structure.
